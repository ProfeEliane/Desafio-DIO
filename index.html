Você sabe o que são Tags?

Tags são identificadores que dizem ao navegador o que é pra ser exibido. Elas são palavras definidas em inglês e aparecem entre os sinais de “menor que” (<) e “maior que” (>)

As Tags devem ser abertas e fechadas para que o navegador saiba onde está definido o início e o fim do elemento. Para se fechar uma Tag é utilizado o sinal /. Existem porém, algumas Tags que não têm fechamento.

Vejamos a seguir algumas das principais Tags:

Índice

  • <h1> a <h6> Define cabeçalhos HTML
  • <hr> Define uma alteração temática no conteúdo
  • <strong> Define um texto em negrito
  • <p> Define um parágrafo
  • <em> Define um texto em itálico
  • <input> Cria um campo texto no arquivo HTML
  • <img> Define uma imagem

Define cabeçalhos HTML

Voltar

As tags <h1> a <h6> são usadas para criar títulos em páginas web, mas o nível de importância é diferente para cada uma, onde h1 tem o nível mais alto e h6, o mais baixo.

Define uma alteração temática no conteúdo

Voltar

A tag <hr> representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibida como uma linha horizontal nos navegadores, mas agora está definida em termos semânticos, em vez de termos de apresentação.

A tag <hr> é considerada uma tag vazia, pois, não necessita fechamento.

Define um texto em negrito

Voltar

A tag <strong> marca um texto considerado importante e o formata em negrito.

Define um parágrafo

Voltar

A tag "p" é utilizada para demarcar um parágrafo. Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento.

Define um texto em itálico

Voltar

A tag <em> marca o texto que tem ênfase e o apresenta na forma de itálico.

Cria um campo texto no arquivo HTML

Voltar

A tag <input> cria uma campo onde o usuário pode inserir dados. O elemento pode ser exibido de várias maneiras, dependendo do atributo type aplicado: text, number ou color, por exemplo.

Define uma imagem

Voltar

A <img> tag é usada para incorporar uma imagem em uma página HTML e apresenta dois atributos obrigatórios:

  • src: Especifica o caminho para a imagem.
  • alt: Especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida.
